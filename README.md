# 🛍️ E-commerce Clickstream Analysis

End-to-end data analytics project focused on understanding user behavior in an e-commerce platform using real clickstream data.

---

## 🎯 Objective

Analyze user interactions to identify:

- Product visibility impact
- User navigation patterns
- Conversion behavior
- Drop-off points in the funnel

---

## 🧠 Business Questions

- Do product positions influence engagement?
- Are discounted products attracting more attention?
- Where do users abandon the website?
- How strong is the conversion funnel?

------
## 🏗️ Project Structure
arch/ → data preparation & cleaning
back/ → business metrics
bi/ → behavioral analysis & insights
notebook/ → exploratory analysis (EDA)

---

## ⚙️ Key Features

### 🔹 Data Cleaning
- Standardized price data
- Removed invalid values
- Added country name mapping

### 🔹 Business Metrics
- Conversion rate (sale vs regular products)
- Interaction analysis by category

### 🔹 User Behavior Analysis
- Sales funnel analysis
- Drop-off and retention rates

---

## 📊 Key Insights

- 📌 Product placement strongly impacts engagement  
- 📌 Top-left positions receive the most clicks  
- 📌 Early-stage drop-off is significant  
- 📌 Discounted products increase interaction rates
